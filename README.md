# Pokémon Backend

This is the backend service for the Pokémon app. It provides REST APIs to fetch Pokémon data from a relational database.

## **Tech Stack**
- Node.js
- Express.js
- PostgreSQL
- Axios (for seeding data)
- Render (for deployment)
